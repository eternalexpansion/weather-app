# Simple Weather App

Simple application for monitoring weather in your city.

## Stack used
- Frontend: Vanilla JavaScript, HTML5, CSS3
- API: OpenWeatherMap

## How to run this app locally:
1. Clone repo: `git clone https://github.com/eternalexpansion/weather-app.git`
2. Open `index.html` in browser or via Live Server VSCode extension
